Explicação das Regras

Regra 1:
Define a cor vermelha para todas as linhas da lista ordenada, ou seja, itens de 1 a 8.
Regra 2:
Define a cor verde para todas as linhas categorizadas pela classe "churras". Mudando os itens 6, 7 e 8 para verde.
Regra 3:
Define a cor cinza para todas as linhas categorizadas pela classe "fruta". Mudando os itens 2, 3, 4 e 5 para cinza.
Regra 4:
Define a cor preta para todos os elementos "li" categorizados com a classe "churras" que seguem diretamente um "li". Mudando os items 6, 7 e 8 de verde para preto.
Regra 5:
Define a cor magenta para todos os elementos "li" categorizados pela classe "churras" e que possuem atributo "draggable="true". Mudando o item 8 de preto para magenta. O atributo "draggable="true" permite que o elemento seja arastável pelo usuário.
Regra 6:
Define a cor azul para os elementos categorizados pelo "id" lip1, lip2 e lip3. Mudando os itens 3 e 4 de cinza para azul. lip1 não definido em nenhum item.
Regra 7:
Define a cor laranja para todos os elementos categorizados pela classe "fruta" com o "id" lip3. Mudando o item 4 de azul para laranja.
Regra 8:
Define a cor roxa para o elemento "li" que é um filho direto de "ol" categorizado pela classe "churras" e que possui o "id" lip5. Não foi definido elementos com id lip5. portanto, nenhum elemento ficou roxo.
