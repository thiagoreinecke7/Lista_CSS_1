Incluindo CSS em Documentos HTML


De acordo com W3schools (2025), existem três maneiras de inserir uma "Style Sheet". São eles:

  • CSS Externo;
  • CSS Interno;
  • CSS Embutido;

CSS Externo

W3schools (2025), com uma "Style Sheet" externa, é possível alterar a aparência de um site inteiro alterando apenas um arquivo. Cada página HTML deve incluir uma referência ao arquivo CSS externo através da tag <link> dentro da seção <head> conforme demonstrado na figura 1.

Figura 1 - Inclusão CSS Externo
Fonte: Elaboração própria.

Uma "Style Sheet" externa pode ser escrita em qualquer editor de texto e deve ser salva com uma extensão .css. O arquivo .css externo não deve conter nenhuma tag HTML.

CSS Interno

Uma "Style Sheet" interna pode ser usada se uma única página HTML tiver um estilo exclusivo. O estilo interno é definido dentro do elemento <style>, dentro do cabeçalho secção <head>, coforme demonstrado na figura 2.

Figura 2 - CSS Interno
Fonte: Elaboração própria.

CSS Embutido

Um estilo embutido pode ser usado para aplicar um estilo exclusivo a um único elemento. Para usar estilos embutidos, adicione o atributo style ao elemento relevante. O style pode conter qualquer propriedade CSS, conforme demonstrado na figura 3.

Figura 3 - CSS Embutido
Fonte: Elaboração própria.

Utilizando Vários Styles Sheet

Ao utilizar vários Styles Sheets é necessário ter cuidado e se atentar a ordem de procedência. Supondo que um CSS externo tenha o estilo da cor verde para o elemento <h1> e um CSS interno tenha o estilo da cor laranja para o mesmo elemento. Se o CSS interno for definido após o link para o CSS externo, os elementos <h1> serão "laranja". Exemplo demonstrado na figura 4.

Figura 3 - Ordem de Procedência
Fonte: Elaboração própria.

No entanto, se o estilo interno for definido antes do vínculo com a folha do CSS externa, os elementos <h1> serão "verde". Exemplo demonstrado na figura 4.

Figura 4 - Ordem de Procedência

É importante lembrar que um estilo embutido tem a prioridade mais alta e substituírá o estilo externo e interno.
